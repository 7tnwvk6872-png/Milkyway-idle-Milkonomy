# Milkonomy v2.1.0 更新日志

**发布日期：2026-07-15**

---

## 神龛数据

导入生活数据时自动识别公会神龛等级，支持五种神龛：力量、节奏、精神、稀有、学者。数据来源于游戏内公会建筑等级，导出脚本自动提取，导入时自动映射填入。

## 导入升级

导入对话框从简单文本框改为完整交互界面。粘贴 JSON 数据后，勾选「数据升级」并通过下拉框选择目标预设，即可一键合并。界面新增提示图标，悬停显示功能说明。

## 数据合并

勾选「数据升级」后导入数据与选中预设自动对比取优。合并规则如下：
- 装备、技能、房子、神龛取两者最高等级
- 社区 Buff 使用最新导入数据
- 预设颜色与封印保持不变
- 不勾选「数据升级」则按原有逻辑创建新预设

适用场景：定期导出自己的最新数据，合并到已有预设中更新装备等级。

## 修复

- 加号按钮在超过 5 个预设后不再消失。
- 导出脚本改为黑名单模式，仅导出生产装备和工具，不再混入战斗装备（武器、盾牌、头盔等）。
- 导出脚本工具装备位不再遗漏。

## 打赏

打赏页面新增 Polokiki 微信收款码。

## 导出脚本 v1.2.0

- 根据游戏域名自动适配中英文界面。
- 更新脚本概述，明确说明支持的数据类型。
- 新增神龛数据导出。
- 修复战斗装备混入问题，改为黑名单过滤。
