# 更新日志

## v2.1.0 — 2026-07-15

### 神龛数据
导入生活数据时自动识别公会神龛等级，支持五种神龛：力量、节奏、精神、稀有、学者。

### 导入升级
导入对话框全新升级：粘贴 JSON → 勾选「数据升级」→ 下拉框选择目标预设 → 一键合并。

### 数据合并
勾选「数据升级」后导入数据与选中预设对比取优：
- 装备、技能、房子、神龛取最高等级
- 社区 Buff 使用最新数据
- 预设颜色与封印保持不变
- 不勾选则创建新预设

### 修复
- 加号按钮超过 5 个预设后不再消失
- 导出脚本仅导出生产装备，不再混入战斗装备
- 导出脚本工具装备位不再遗漏

### 打赏
新增 Polokiki 微信收款码。

### 导出脚本 v1.2.0
- 根据游戏域名自动适配中英文界面
- 更新脚本概述，明确功能说明
